# AI tool usage

This project was built with the help of AI tools, used for different parts of the workflow:

- **ChatGPT** — brainstorming and planning: early ideation, architecture decisions, and general problem-solving discussions.
- **GitHub Copilot** — inline code completion while hand-writing code in the IDE.
- **Claude Code** — hands-on implementation: feature work, refactors, bug fixes, the UI/UX redesign, test fixes, and this documentation.

All AI-assisted code was reviewed and tested before being committed.
